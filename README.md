FRAME FOLK PHOTOGRAPHY WEBSITE
==============================

Complete Photography Portfolio Website with Admin Panel

FILES INCLUDED:
1. index.html - Main website
2. styles.css - All styling
3. script.js - JavaScript functionality
4. admin-panel.html - Admin dashboard
5. README.txt - This file

HOW TO USE:
1. Save all files in the same folder
2. Open index.html in a web browser
3. For admin access, click "Admin" in navigation or open admin-panel.html

FEATURES:
- Responsive design (mobile-friendly)
- Portfolio gallery with filtering
- Contact form with data storage
- Testimonial slider
- Admin dashboard to view inquiries
- Professional photography aesthetic

ADMIN FEATURES:
- View all customer inquiries
- Export data as JSON
- Clear all data
- Statistics dashboard

TECHNOLOGY:
- Pure HTML, CSS, JavaScript
- No backend required
- Uses localStorage for data storage
- Font Awesome icons
- Unsplash images

CUSTOMIZATION:
- Edit styles.css to change colors
- Update content in index.html
- Modify portfolio items in script.js
- Change contact information

SUPPORT:
This is a complete frontend solution. For backend integration, additional development would be needed.

Enjoy your new photography website!
